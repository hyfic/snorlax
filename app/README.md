# SNORLAX APP

Client for snorlax.

> Built using, Tauri, Rust, Typescript & React.

## SETUP

Setup [Tauri](https://tauri.app/v1/guides/getting-started/prerequisites) & Rust

Install packages

```bash
$ npm install
## Or using pnpm
$ pnpm install
```

Run client

```bash
$ npm run tauri dev
## Or using PNPM
$ pnpm tauri dev
```
